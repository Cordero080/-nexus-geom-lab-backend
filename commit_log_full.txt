ðŸŽµ useAudioAnalyzer.js - Audio Frequency Analysis
--------------------------------------------------
WHAT IT DOES:
- Analyzes real-time audio to extract frequency data (bass, mids, highs)
- Breaks audio into frequency bands to drive visual effects
- Normalizes audio data to 0-1 range for consistent animations

NEW FEATURES:
- Split bass into sub-bass (20-60Hz - deep kicks) and mid-bass (60-250Hz - toms/bass guitar)
- Weighted combination: sub-bass gets 70% influence (kicks hit harder visually)
- Sharp highs (2000-8000Hz) now focus on transient sounds (snare, hi-hat, claps)
- Filters out sustained high frequencies to emphasize percussive hits

HOW IT WORKS:
1. Web Audio API captures audio stream from user's microphone/audio input
2. AnalyzerNode converts audio to frequency data using FFT (Fast Fourier Transform)
3. getFrequencyRangeAverage() samples specific Hz ranges and normalizes them
4. Returns { bass, mids, highs, overall } object that components can use

COMMUNICATES WITH:
- Any component that imports useAudioAnalyzer() hook
- Drives particle systems, color changes, animations based on music
