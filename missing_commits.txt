ðŸŽµ useAudioAnalyzer.js - Audio Frequency Analysis
--------------------------------------------------
WHAT IT DOES:
- Analyzes real-time audio to extract frequency data (bass, mids, highs)
- Breaks audio into frequency bands to drive visual effects
- Normalizes audio data to 0-1 range for consistent animations

NEW FEATURES:
- Split bass into sub-bass (20-60Hz - deep kicks) and mid-bass (60-250Hz - toms/bass guitar)
- Weighted combination: sub-bass gets 70% influence (kicks hit harder visually)
- Sharp highs (2000-8000Hz) now focus on transient sounds (snare, hi-hat, claps)
- Filters out sustained high frequencies to emphasize percussive hits

HOW IT WORKS:
1. Web Audio API captures audio stream from user's microphone/audio input
2. AnalyzerNode converts audio to frequency data using FFT (Fast Fourier Transform)
3. getFrequencyRangeAverage() samples specific Hz ranges and normalizes them
4. Returns { bass, mids, highs, overall } object that components can use

COMMUNICATES WITH:
- Any component that imports useAudioAnalyzer() hook
- Drives particle systems, color changes, animations based on music

ðŸŒŒ useNebulaParticles.js - Symphonic Particle Motion
-----------------------------------------------------
WHAT IT DOES:
- Creates 200 floating particles that move in synchronized orbital patterns
- Divides particles into 3 groups that move like a symphony orchestra
- Each group has unique motion choreography inspired by Beethoven's 9th Symphony

THE 3 ORCHESTRA SECTIONS:
1. STRINGS (Group 0 - Inner Orbit):
   - Smooth flowing melody like violins playing 'Ode to Joy'
   - Graceful sinusoidal motion with harmonic intervals
   - Tight 8-unit orbit with gentle vertical breathing
   - Slightly more visible (opacity 0.2)

2. BRASS (Group 1 - Middle Orbit):
   - Bold punctuated phrases like horns and trumpets
   - Staccato (stepped) motion with dramatic swells
   - Wide 22-unit orbit with vertical stratification
   - Crescendo/decrescendo opacity changes (forte/piano)

3. TIMPANI (Group 2 - Outer Orbit):
   - Rhythmic pulses like percussion heartbeat
   - Radial bursts that expand/contract on 4/4 beat
   - Far 40-unit orbit with accented 4th beat (boom-boom-boom-BOOM)
   - Sharp attack/decay opacity

MOTION MATHEMATICS:
- Uses sin/cos for circular orbits and figure-8 paths
- Phase offset (unique per particle) prevents clustering
- Multiple frequencies combined (t * 0.25, t * 2, t * 0.5) create harmony
- Squared sine waves (Math.pow) create dramatic crescendos

RENDERING:
- depthTest: false - particles render in front of everything
- depthWrite: false - don't block other particles
- renderOrder: 0 - render after opaque objects
- Wireframe particles mixed with solid for texture

COMMUNICATES WITH:
- Three.js scene (adds meshes to scene graph)
- Animation loop (updates position/opacity every frame)
- No backend communication - pure client-side visual effect
